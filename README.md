# WanAndroid
>书已经看得够多了，我需要积累一些实战经验
并且有些未能理解通透的知识也需要做项目来巩固
于是就有了这个APP,学长们当年的寒假作业

## 目标
- 积累项目经验
- 实践MVVM架构
- 开发出美观的UI

## 目前实现的功能

- 首页RecyclerView使用Paging组件分页加载，防止一次加载过多文章造成流量的浪费

- 登录/注册功能 （使用给OKHttpClient注册Interceptor的方式缓存Cookie）

- MVVM架构骨架
